# ZDocs
一款基于 PHP 开发的轻量级文档系统，采用无后台设计，所有交互均在前台完成。

![image](https://github.com/user-attachments/assets/00f462b7-0bab-4602-86ac-260ed0339a0a)

## 功能和特性
* 轻到不能再轻量化，总共就 9 个 PHP 文件
* 内容支持 Markdown 和 HTML，采用前端渲染 Markdown
* 支持多分类、子分类
* 支持多用户协作
* 支持文章和分类排序
* 支持明暗主题切换
* 密码使用 BCRYPT 散列储存
